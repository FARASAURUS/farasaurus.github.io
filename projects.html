Projects

Flappy Bird in C++

Selama menempuh mata kuliah Dasar-Dasar Pemrograman, saya berhasil mengembangkan sebuah game sederhana berbasis C++ yang terinspirasi dari Flappy Bird. Project ini merupakan implementasi dari pemahaman saya mengenai konsep-konsep fundamental pemrograman, seperti: Struktur Data, Kontrol Alur Program, Manajemen Input/Output, Dasar Game Logic.

Pahlawan Padang Pasir

Dalam mata kuliah Pemrograman Berorientasi Objek, saya membuat game Pahlawan Padang Pasir menggunakan JavaFX untuk menerapkan konsep OOP seperti enkapsulasi, inheritance, dan polimorfisme. Game ini menampilkan perjalanan seorang pahlawan di gurun pasir dengan mekanisme pergerakan, serangan, dan penyimpanan progres menggunakan file handling. Saya menggunakan FXML untuk desain UI dan mengembangkan logika permainan dengan Java. Proyek ini melatih kemampuan saya dalam merancang kode terstruktur dan mengelola dependencies.

Kingz Barbershop

Dalam menyelesaikan tugas akhir mata kuliah Rekayasa Perangkat Lunak, saya bersama lima anggota tim lainnya mengembangkan desain UI/UX aplikasi barbershop menggunakan Figma dengan pendekatan Agile Scrum. Proyek ini bertujuan untuk menciptakan solusi digital yang meningkatkan pengalaman pelanggan dalam memesan layanan potong rambut secara online, sekaligus menerapkan prinsip-prinsip rekayasa perangkat lunak mulai dari analisis kebutuhan hingga evaluasi produk.
